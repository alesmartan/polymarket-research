### Content Tone & Voice Guidelines

**Brand Voice Characteristics:**

| Attribute | Description | Example |
|-----------|-------------|---------|
| **Authoritative** | Expert but approachable | "The data shows..." not "We think maybe..." |
| **Clear** | Simple explanations | No jargon without explanation |
| **Engaging** | Active, conversational | "You can profit from..." not "Users are able to..." |
| **Trustworthy** | Transparent, balanced | Acknowledge risks alongside opportunities |
| **Energetic** | Excited but not hype-y | Enthusiasm without pump language |

**Writing Guidelines:**
- Use active voice
- Keep sentences concise (avg 15-20 words)
- Break up text with subheadings every 300 words
- Include data and specific examples
- Always include clear CTAs
- Avoid regulatory red flags (no investment advice)

### On-Page SEO Checklist

**For Every Blog Post:**

- [ ] Primary keyword in title (H1)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in meta description
- [ ] 2-3 secondary keywords naturally included
- [ ] Primary keyword in URL slug
- [ ] Primary keyword in at least one H2
- [ ] Alt text on all images with keywords
- [ ] Internal links to 3+ related articles
- [ ] External links to 2+ authoritative sources
- [ ] Schema markup (Article, FAQ, How-To)
- [ ] Meta description 150-160 characters
- [ ] Title tag 50-60 characters
- [ ] Content length matches top-ranking competitors

- [ ] Site loads in <3 seconds
- [ ] Mobile-responsive design

- [ ] XML sitemap submitted
- [ ] Robots.txt optimized
